Aufgabe 16 - Suche

            $('#btnAbonnieren').show();
            $('#divAbonniert').hide();
        });
    - Folgendes hinzu:
        //Suche - Funktion
        $('#Suche').on('input', function () {
                var suchwert = $(this).val();                
                $('#Galerie img').each(function () {
                if ($(this).attr('src').indexOf(suchwert) != -1) {
                    $(this).show();
                } else {
                    $(this).hide();
                }
            });
        });
    - Gebt etwas in das Suchfeld ein
    - Nun gebt einen image namen ein
